  1. position의 기초

    기본적인 포지션을 배워보았다.

    2022.05.19 display는 어떻게?, position은 어디에? 
                오늘 배운 position은 static, relative, absolute, fixed!
            여기서 좌표를 가지지않는 것은 static 
            스크롤 바가 없어지게 하려면 fixed
            박스를 가운데 오게 하는 방법
            1. 저번 시간에 배운 marign: 0 auto;
            2. 오늘 배운 position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
            이것들은 공식이다.

            또 하나 더, transform: translateX(-50%); -> 자기 자신의 사이즈의 50%를 이동 하는 것.
            transform은 두번 쓰지 아니한다.

    느낀점 : 수업을 들으면서 따라 했던 것과 달리 혼자서 해보면 생각보다 어떤 식으로 포지션이 잡히는 지 감이 잘 안 올 것 같다. 대략적으로 어떤 식으로 실행이 되는 지는 감이 잡혔다. 대학생 때 야매로 하던 것과는 많이 다른 것 같다. 진작 학교에서 알려줬더라면 더 흥미롭게 했을 것 같다.
